# [StyleGen]

## 📝 Descrição
UI Style Generator é uma ferramenta inteligente projetada para otimizar a etapa de design no desenvolvimento web. O sistema gera combinações harmônicas de tipografia e paletas de cores acessíveis, permitindo que desenvolvedores visualizem a identidade visual do site antes mesmo de escreverem a primeira linha de CSS.

## ⚙️ Parâmetros Utilizados
Model: Utilizamos o gpt-3.5-turbo (ou gpt-4) por sua alta velocidade de processamento e excelente compreensão de harmonia visual.

Prompt System: Definimos o papel da IA como um "Designer UI Expert" para garantir que as combinações de fontes e cores sigam princípios de acessibilidade e contraste.
## 🌡️ Comparação de Temperatura
Temperatura 0.2 (Baixa): As sugestões foram focadas em padrões seguros e profissionais. A IA tendeu a repetir combinações clássicas (ex: Fontes Inter/Roboto com paletas monocromáticas azuladas). Ideal para sites corporativos.

Temperatura 1.2 (Alta): A IA tornou-se muito mais ousada e experimental. Surgiram fontes decorativas e paletas vibrantes/complementares (ex: Neon com fundos escuros). Ideal para portfólios criativos e projetos artísticos.
